# 💍 Online Wedding Invitation

![Status](https://img.shields.io/badge/status-active-success.svg)
![Node](https://img.shields.io/badge/node.js-18.x-green)
![License](https://img.shields.io/badge/license-MIT-blue)

Website undangan pernikahan online modern dengan fitur **RSVP (konfirmasi kehadiran)** dan **buku tamu digital**. Dibangun dengan Node.js, Express, dan vanilla JavaScript.

## ✨ Fitur Unggulan

- 📱 **Responsive Design** - Tampil sempurna di HP, tablet, dan desktop
- 💌 **Undangan Digital** - Informasi lengkap pasangan, waktu, dan lokasi acara
- ✅ **RSVP Online** - Tamu bisa konfirmasi kehadiran dengan mudah
- 💬 **Buku Tamu** - Tamu bisa meninggalkan ucapan dan doa
- 📊 **Real-time Stats** - Lihat statistik kehadiran otomatis
- 🔒 **Data Privacy** - Data tamu tersimpan aman di file JSON

## 🛠️ Teknologi yang Digunakan

| Teknologi | Fungsi |
|-----------|--------|
| Node.js | Backend JavaScript runtime |
| Express.js | Web framework untuk API |
| Vanilla JS | Frontend interaktif (tanpa framework) |
| HTML5 & CSS3 | Struktur dan styling |
| JSON File | Database ringan (tanpa setup) |

## 🚀 Cara Menjalankan

### Prasyarat
- Node.js (versi 14+)
- npm (biasanya sudah termasuk Node.js)

### Langkah-langkah

```bash
# 1. Clone repository ini
git clone https://github.com/TAHUKU/online-wedding-invite.git

# 2. Masuk ke folder proyek
cd online-wedding-invite

# 3. Install dependencies
npm install

# 4. Jalankan server
node server.js

# 5. Buka browser dan akses
http://localhost:3000
